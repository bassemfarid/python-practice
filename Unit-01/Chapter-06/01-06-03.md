# Unit 1 Lesson 6 Question 3

## Instructions
Create an add-word program. Take in a sentence that is missing a word. Take in a word that you need to insert into the sentence. Lastly, ask the user the index location of the space where they want to insert the word. Output the sentence with the inserted word.

## Sample Input

```
I do eat chickens
not
4
```

## Output for Sample Input

```
I do not eat chickens
```
