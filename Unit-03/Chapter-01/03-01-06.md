# Unit 3 Chapter 1 Question 6 - Sum of 1 to X

Create a program that accepts an integer from the user (above 1) and calculates the sum of all integers between 1 and the given integer (including the given integer and 1).

## Input Specification  
The input consists of a single integer, $C$, where 1 < $C$

## Output Specification  
The output consists of a single integer, representing the sum of all of all numbers between 1 and the given integer (inclusive)

## Sample Input
```
6
```

## Output for Sample Input
```
21
```

## Explanation of Output for Sample Input
If the numbers `1` through `6` are added, the sum is `21`.
